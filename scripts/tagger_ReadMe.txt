AUTOMATIC1111/stable-diffusion-webui 的脚本，用于提供一种将列表中的标签快速添加到提示符的方法

它能做什么
该脚本将添加将标签添加到单独文本框的功能。这些将自动添加到提示的末尾。您可以手动添加标签，也可以选中下面出现的任何复选框。您还可以选中“添加注意括号”以使用 AUTOMATIC1111 的新注意括号添加术语：(term:1.1)，其中 1.1 是默认值，因此添加了它。您可以手动编辑这些。检查文档以获取对此的解释。您可以在此文本框中输入任何内容，当您单击“生成”时，其内容将添加到提示的末尾。

安装
从https://github.com/AUTOMATIC1111/stable-diffusion-webuiscripts将文件夹中的文件克隆或下载到文件夹中scripts
在 webui 的“脚本”下拉列表中选择“标记器”。
特征
可定制的标签文件。定义您自己喜欢的标签并删除您不喜欢的标签。
(greg rutkowski:1.1)您可以通过选中“带注意括号添加”复选框来添加/删除带有注意括号的标签（例如）。现在添加的任何标签都会有它们。
#通过添加 a作为行中的第一个字符来排除某些标签显示在列表中。
文件中的标签favorites.txt始终显示在列表顶部
unused_tags不会添加文件夹中的文件
定制
我已经添加了一些带有标签的文本文件。这些是从文件夹中添加的，该文件夹/scripts/tagger/包含带有标签的文本文件。您可以通过创建新文本文件并在其中添加自定义标签或修改提供的文本文件来以任何方式组织标签。重新加载 web ui 时将加载更改或新文件。每行都是一个新标签。标签按照您使用的文件名进行组织。我已经包括了一些：

文件	描述
艺术家.txt	SD认可的所有艺术家的名单。取自 Manav Mashruwala 的伟大谷歌表：https ://docs.google.com/spreadsheets/d/1SRqJ7F_6yHVSOeCi3U82aA448TqEGrUlRrLLZ51abLg
相机.txt	与相机相关的术语，例如焦距、景深等。
灯光.txt	与场景照明有关的术语，例如刺眼的照明、戏剧性的照明、阴影等。
样式.txt	图像风格术语，如油画、动漫、黑白等。
网站.txt	网站名称，例如 artstation 或 pixiv。
颜色.txt	涉及颜色的描述符，例如鲜艳度
环境.txt	环境描述的标签，例如比例
3d-renders.txt	涉及 3D 渲染外观的标签，例如体积雾或 cgi
